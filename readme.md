## Installation

1. Open Acode App, and search for "Catppuccin" in plugin store.
2. Install it!
3. Done :)


## Usage

Once installed, you can switch between flavors in Acode settings(`Acode App > Settings > Themes`). Each flavor has its own personality and color palette:

- **Latte** ðŸŒž - A light theme with soft, pastel colors perfect for well-lit environments.
- **FrappÃ©** ðŸª´ - A dark theme with a muted color palette, excellent for coding in moderate lighting.
- **Macchiato** ðŸŒº - A rich dark theme with vibrant yet gentle colors, ideal for low-light settings.
- **Mocha** ðŸŒ¿ - A deeper dark theme with intense, warm colors for a cozy night-time coding experience.
